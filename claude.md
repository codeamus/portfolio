## 📋 FASE 6: INTEGRACIÓN DE TASTE-SKILL

### 6.1 Cómo Usar Taste-Skill

El archivo SKILL.md de taste-skill define 3 settings clave:

```
DESIGN_VARIANCE: "moderate" (layouts creativos pero coherentes)
MOTION_INTENSITY: "high" (animaciones impactantes)
VISUAL_DENSITY: "low" (premium, mucho whitespace)
```

**Aplicar para cada componente:**
- Leer la guía taste-skill
- Generar componentes con principios premium
- Rechazar cualquier diseño "genérico" o "slop"
- Priorizar animaciones suaves y transiciones

### 6.2 Principios Clave

✅ **Spacing**: 8px grid, mucho whitespace
✅ **Typography**: Contrast alto, jerarquía clara
✅ **Motion**: Purposeful, no distracting
✅ **Color**: Paleta consistente, acentos estratégicos
✅ **Imagery**: Alta resolución, relevante
✅ **Micro-interactions**: Detalles que sorprenden
