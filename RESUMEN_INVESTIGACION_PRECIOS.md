## 📈 ESTRATEGIA DE VENTA

### Segmentación de Clientes

```
STARTUP/EMPRENDEDOR
├─ Presupuesto: < $1,000,000
├─ Necesidad: MVP rápido
└─ Solución: Landing Page Premium ($500K) → Sitio Web ($1.2M)

PYME LOCAL
├─ Presupuesto: $1,000,000 - $3,000,000
├─ Necesidad: Sitio web + posible app
└─ Solución: Sitio Profesional + E-commerce ($1.5M - $2.2M)

EMPRESA MEDIANA
├─ Presupuesto: $3,000,000 - $15,000,000
├─ Necesidad: Web + App completa
└─ Solución: Suite Completa ($10M - $15M)

AGENCIA/RESELLER
├─ Presupuesto: Sin límite
├─ Necesidad: Servicios white-label
└─ Solución: Presupuesto especial, contrato anual
```

### Recomendación de Venta

1. **Fase 1: Calificación**
   ```
   Pregunta clave: "¿Cuál es tu presupuesto aproximado?"
   
   <$500K → Landing Page
   $500K-$1.5M → Sitio Web
   $1.5M-$5M → Sitio + App Básica
   >$5M → Suite Completa
   ```

2. **Fase 2: Propuesta**
   ```
   No presentes 1 opción, presenta 3:
   
   - STARTER (más económico)
   - POPULAR (recomendado) ✓ Este
   - ENTERPRISE (premium)
   ```

3. **Fase 3: Cierre**
   ```
   "Podemos comenzar con Landing Page ahora
   y luego escalar a Sitio Web y App"
   
   Efecto: Hace sentir menos riesgo al cliente
   ```
