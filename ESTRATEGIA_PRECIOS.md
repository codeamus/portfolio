## 🎯 DECISIÓN #1: ¿Mostrar precios o no?

### OPCIÓN A: Mostrar rango abierto
```
Ventaja: Transparencia, confianza, menor fricción
Desventaja: Clientes buscarán lo más barato

EJEMPLO:
"Landing Pages desde $350,000 CLP"
"Sitios Web desde $1,200,000 CLP"
"Apps Móviles desde $5,000,000 CLP"
```

### OPCIÓN B: No mostrar precios (mostrar "Contacta para presupuesto")
```
Ventaja: Más conversión de contactos, leads cualificados
Desventaja: Menos transparencia, puede ahuyentar clientes

EJEMPLO:
"Presupuesto personalizado según tu proyecto"
"Contacta para una cotización sin compromiso"
```

### RECOMENDACIÓN PARA CODEAMUS:
**OPCIÓN A (Mostrar rango)**

Razones:
- Eres freelancer, no agencia grande (transparencia importante)
- Clientes en Chile ESPERAN ver precios
- Aumenta credibilidad vs competencia que esconde precio
- Atrae el público correcto desde el inicio

## 💰 DECISIÓN #2: ¿Cómo estructurar los precios?

### ESTRUCTURA 1: Precios por servicio (Recomendado)
```
Landing Page: desde $350,000
Sitio Web: desde $1,200,000
E-commerce: desde $1,200,000
App Móvil: desde $5,000,000

Ventaja: Fácil de entender, cliente sabe qué esperar
```

### ESTRUCTURA 2: Precios por paquete
```
Paquete Básico (Landing): $350,000
Paquete Profesional (Sitio): $1,200,000
Paquete Premium (Web+App): $14,500,000

Ventaja: Crea urgencia, bundling effect
```

### ESTRUCTURA 3: Precios por nivel de complejidad
```
Proyecto Simple: $500,000-$1,000,000
Proyecto Intermedio: $1,000,000-$3,000,000
Proyecto Complejo: $3,000,000+

Ventaja: Flexible, cliente entiende cómo escala
```

### RECOMENDACIÓN:
**ESTRUCTURA 1 + ESTRUCTURA 3**

Combina ambas:
- Servicio (qué es)
- + Complejidad (cuánto cuesta)

Ejemplo:
```
Landing Page
├── Básica: $350,000 (simple)
└── Premium: $650,000 (con animaciones + CRM)

Sitio Web
├── Profesional: $1,200,000 (estándar)
└── Avanzado: $1,800,000 (reservas + marketing)
```

## 🎯 PSICOLOGÍA DE PRECIOS

### Técnica 1: Anclaje
```
Mostrar precio ALTO primero, luego bajar

❌ INCORRECTO:
"Desde $350,000"

✅ CORRECTO:
"Apps Móviles desde $5,000,000
Sitios Web desde $1,200,000
Landing Pages desde $350,000"

Efecto: El cliente piensa "$350K es barato"
```

### Técnica 2: Números pares vs extraños
```
❌ $1,200,000 (suena caro)
✅ $1,150,000 o $1,199,000 (suena más asequible)

Pero en tu caso, mantén números redondos:
✅ $1,200,000 (confianza)
```

### Técnica 3: Mostrar valor, no solo precio
```
❌ "Sitios web: $1,200,000"

✅ "Sitios Web Profesionales
   $1,200,000
   ✓ 5-10 páginas
   ✓ Diseño premium
   ✓ Blog integrado
   ✓ SEO técnico
   ✓ Dominio + hosting 1 año
   ✓ 1 mes de soporte"
```

### Técnica 4: "Desde $X" siempre
```
✅ "Desde $350,000"
   (Cliente sabe que puede ser más, pero se siente accesible)

✅ "Desde $5,000,000"
   (No asusta, es "desde")

❌ "Cuesta $5,000,000"
   (Sonará fijo y caro)
```

### Técnica 5: Desglosaes
```
✅ "$1,200,000 por sitio profesional"
   = $240,000 por página (si son 5)
   = $40,000 por día (si son 30 días)
   
Efecto: El cliente ve que es "razonable por día"
```

## 📋 RESPUESTAS A OBJECIONES DE PRECIO

### "¿Por qué es tan caro?"

**RESPUESTA:**
```
"No somos la opción más barata, pero:

✓ Diseño custom (no template)
✓ SEO optimizado desde día 1
✓ Soporte 1 mes incluido
✓ Código limpio y mantenible
✓ Performance optimizado (velocidad)
✓ Seguridad y actualizaciones

Si buscas lo más económico, hay opciones.
Si buscas relación calidad-precio, aquí estoy."
```

### "¿Puedo pagar menos?"

**RESPUESTA:**
```
"Sí, opciones:

1. Reducir scope:
   - 1 página en lugar de 5
   - Solo web, sin app
   - Funcionalidades básicas

2. Plazo flexible:
   - Más tiempo = menos costo
   - Hablo con mi agenda

3. Mantenimiento:
   - Haz cambios tú mismo (solo soporte)
   - Ahorra en mantenimiento

¿Cuál te interesa explorar?"
```

### "Un competitor cobra menos"

**RESPUESTA:**
```
"Puede ser. Consideraciones:

✓ Verificar: ¿Incluye dominio + hosting?
✓ ¿Cuánto soporte está incluido?
✓ ¿Es custom o template?
✓ ¿Quién hace mantenimiento post-launch?
✓ ¿SEO desde cero o básico?

Muchas veces lo barato cuesta caro después.
Podemos hacer una auditoría de propuestas
y encontrar el mejor balance.

Quiero que tengas el mejor ROI, no el precio más bajo."
```
